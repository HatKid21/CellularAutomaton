# Cellular Automaton

---

A simple showcase of Conway's Game of Life written in Java using raylib.

It uses [Raylib-J](https://github.com/CreedVI/Raylib-J) library.

# Bindings

Default config cosists of :
- Q - quit game
- W - speed up simulation
- S - speed down simulation
- R - randomize cells
- SPACE - pause the simulation
- N - simulate next step (works only on pause)
- C - clear the grid
